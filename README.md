# 🌡️ Calculadora Inteligente de IBUTG Médio


\

## 📖 Sobre o projeto

A **Calculadora Inteligente de IBUTG Médio** foi desenvolvida como um projeto de estudo e portfólio na área de **Segurança e Saúde no Trabalho (SST)**.

O objetivo da ferramenta é automatizar o cálculo do **Índice de Bulbo Úmido Termômetro de Globo Médio (IBUTG Médio)**, reduzindo erros manuais e tornando o processo de avaliação mais rápido, organizado e confiável.

Este projeto foi desenvolvido utilizando recursos nativos do Excel, como tabelas inteligentes, fórmulas e validações automáticas.

---

## 🎯 Objetivos

* Automatizar o cálculo do IBUTG Médio;
* Reduzir erros de preenchimento;
* Facilitar a organização das medições;
* Criar uma ferramenta prática para estudos e futuras aplicações profissionais;
* Desenvolver habilidades em Excel aplicadas à Segurança do Trabalho.

---

## ✨ Funcionalidades

* ✅ Cadastro ilimitado de medições;
* ✅ Tabela Inteligente do Excel;
* ✅ Cálculo automático do IBUTG Médio;
* ✅ Contagem automática das medições;
* ✅ Soma automática do tempo de exposição;
* ✅ Validação das linhas preenchidas;
* ✅ Interface simples e intuitiva.

---

## 📐 Metodologia

O cálculo do IBUTG Médio é realizado utilizando a média ponderada pelo tempo de exposição:

[
IBUTG_{médio}=\frac{\sum(IBUTG_i\times Tempo_i)}{\sum Tempo_i}
]

A ferramenta considera todas as medições inseridas na tabela inteligente, permitindo ao usuário adicionar quantas linhas forem necessárias.
